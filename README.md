# Ripple Button Hover Effect 

A smooth ripple hover effect built using HTML, CSS, and JavaScript.
The ripple originates from the exact mouse position using CSS variables.

## Features
- Mouse-position-based ripple effect
- CSS pseudo-elements and transitions
- Dynamic CSS variables via JavaScript
- Lightweight and responsive

## Tech Stack
- HTML
- CSS
- JavaScript

## What I Learned
- Using CSS variables with JavaScript
- Mouse position calculations
- Pseudo-elements (`::before`)
- UI interaction effects

## How to Run
Open `index.html` in the browser or use Live Server.
